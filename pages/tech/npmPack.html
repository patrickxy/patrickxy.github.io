# 修改 packge.json 配置

{
  "name": "patrick-utils",
  "private": false,
  "version": "0.1.4",
  "license": "MIT",
  "scripts": {
    "serve": "vue-cli-service serve",
    "build": "vue-cli-service build",
    "lint": "vue-cli-service lint",
    "package": "vue-cli-service build --target lib --name index     ./src/utils/index.js"
  },
  "main": "./dist/index.js"
}

配置说明

  1. name npm 的包名，发布成功后可以在 npm 官网 上搜索

  2. private->false 设置为公开

  3. version 版本号 每次发布版本号必须更新

  4. 添加 license MIT

  5. scripts 中添加编译项 "package": "vue-cli-service build --target lib --name index ./src/utils/index.js"

    --targer lib 代表打包成库文件

    --name index 打包生成文件的名称

    ./src/utils/index.js 代码的入口文件地址

  6. 添加 main: "/dist/index.umd.min.js" 入口文件

npm 包发布流程

  1. 查看当前的 npm 源 npm config get registry npmjs.com
  2. npm login 然后输入账号密码 登录
  3. npm publish 发布，成功后命令行会有提示信息
  4. npm unpublish 撤回 npm 包
